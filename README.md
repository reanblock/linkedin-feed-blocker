# LinkedIn Feed Blocker

A neat, simple and compact LinkedIn feed blocker.

## Installation 

Steps to install in Chrome based browsers is:

1. Clone this repo to your machine.
1. Open Chrome and go to `chrome://extensions`.
1. Toggle on Developer mode in the top-right corner.
1. Click Load unpacked, and select your `linkedin-feed-blocker` folder. 
1. Refresh your LinkedIn feed tab and the feed will be replaced with a small "Feed hidden" message.

Note this extension should only block the feed - all other features should remain intact so that you can work on important stuff like messaging your contacts, updating your profile and building your network.
